명령어의 종류

1. ls: 파일 목록 확인
	옵션(파라미터)
	l: list / 권한, 크기, 생성시
		- 맨 앞에 d가 붙어있다면, directory 그렇지 않다면, 파일

	a: all 모든 파일을 보여줌(감춰진 파일('.'으로시작하는 파일))

	S: sort 큰 파일을 위로 보여줌

	필요시 man으로 옵션을 확인한다.

2. pwd: 현재 위치 확인

3. mkdir "파일이름" : 디렉토리 생성

4. cd: change directory 경로를 이동하는 명령어
	cd .. : 현재 디렉토리의 부모로 이동(상대경로 이동)

	cd / : root 디렉토리로 이동(절대 경로)
		cd /dir1/dir2 와같이 사용할 수 있음

5. clear: cmd 화면을 비움

6. rm: remove 파일 삭제 명령어
	rm 파일명.확장자
	-r 파라미터: (recursive)디렉토리 삭제 ex) rm -r 디렉토리명


7. --help 파라미터: 해당 명령어의 파라미터 설명

8. man: manual 명령어 관한 자세한 설명
	man 명령어
	특정 단어찾기: /찾을단어 -> n키로 넘어가기 -> q로 탈출

9. vi 명령어
	이동
		- gg 문서 맨 앞 이동
		- G 문서 맨 뒤 아래 이동
		- ^ 문장 맨 앞 이동
		- $ 문장 맨 뒤 이동 
	저장
		- :w 저장
		- :q 닫기
		- :q! 저장하지 않고 닫기
		- :숫자 지정한 줄 번호로 이동
